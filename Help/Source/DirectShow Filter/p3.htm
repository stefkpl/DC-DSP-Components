The Winamp2 DSP Plugins Form. The usage is the same as with the Visual Plugins. The Enabled CheckBox (1) will Enable/Disable the processing of these Plugins. I also added a way to to hide the automatically loaded Options Window(s) some Plugins show. The ComboBox (2) holds a List of these Windows. To add a Window to this List do a LeftClick on the Binoculars (3), hold the MouseButton and move it to that Window that should be hidden. The next Time the Filter loads it will hide this Window. To remove a Window from the List, select it from the ComboBox and hit the delete Button.
