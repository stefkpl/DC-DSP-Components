The Winamp2 Visual Plugins Form. If you worked with them before, you should know what to do here. The first Time the Filter runs, it tries to retrieve the Plugins Directory from the Registry. This works only if you have Winamp installed. Otherwise you have to choose the Plugins Directory manually. The Interval (1) specifys how often the Filter will push Data to the Plugin. This shouldn't be set too low. A value of 40ms is actually enough. Autostart (2) will start the last chosen Visual Plugin automatically the next time the Filter is created. The Red Cross (3) will remove the Plugin List. This will increase the loading Time of the Filter a bit.

A small Note: I am aware that not every Plugin is working within this Filter. Some Plugins are searching for the Winamp Window (that doesn't exist), others don't like the Threaded creation ...
If a Plugin isn't working, too bad ... try contacting the Author if he is going to make it usable within DirectShow Applications.
