This is the main Window where you setup the DSP's. By doing a RightClick on the Listbox (1) you can add a DSP or DMO Filter. From this Menu you´re also able to save the current Settings as a Preset, which are shown in the Menu and in the ComboBox (2). The Save Button will Quick Save the current Settings under the selected Preset Name. If no Preset is chosen, the Filter will ask for a Name.




After you inserted a DSP, you can show its Property Form by doing a LeftClick on it in the List (3). Now you´re able to setup the DSP specific Parameters. The Filter will automatically save the current selected and enabled DSP's and the DSP specific settings when reloading it. The sliders left (4) and right (5) at the Spectrum Display (6) controls the Spectrum Falloff Speed (left) and the Amplitude (right). The Spectrum itself can be disabled by doing a LeftClick on it. The Spectrum Colors can be selected by doing a RightClick on it.
